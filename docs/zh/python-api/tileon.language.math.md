# tileon.language.math

## 数学函数

### 指数函数

- `exp`: 指数函数 `e^x`
- `exp2`: 指数底数 2 `2^x`
- `rsqrt`: 倒数平方根 `1/sqrt(x)`

### 对数函数

- `log`: 自然对数 `ln(x)`
- `log2`: 对数底数 2

### 三角函数

- `cos`: 余弦函数
- `sin`: 正弦函数

### 幂和根

- `sqrt`: 平方根
- `sqrt_rn`: 四舍五入平方根

### 算术运算

- `fma`: 融合乘加 `a * b + c`
- `fdiv`: 快速除法
- `div_rn`: 四舍五入除法
- `umulhi`: 乘法高位

### 取整操作

- `floor`: 向下取整
- `ceil`: 向上取整

### 其他

- `abs`: 绝对值
- `erf`: 误差函数
