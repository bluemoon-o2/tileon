# 欢迎使用 Tileon 文档！

Tileon 是一个用于并行编程的语言和编译器。它旨在提供一个基于 Python 的编程环境，用于高效编写能够在现代 GPU 硬件上以最大吞吐量运行的自定义 DNN 计算内核。

## 入门

- 按照 [安装指南](getting-started/installation.md) 进行安装。
- 查看 [教程](getting-started/tutorials/index.md) 学习如何编写您的第一个 Tileon 程序。
